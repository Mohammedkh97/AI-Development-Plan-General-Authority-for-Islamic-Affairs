تنبيه الـ Guardrail — مسألة خلافية معاصرة (تم إيقاف النشر الآلي)

هذه المسألة فيها خلاف معاصر بين المجامع الفقهية. استرجع النظام أقوالاً متعددة بنسب ثقة متقاربة. يوصي النظام بمراجعة المفتي البشري لاختيار الرأي المعتمد لدى الهيئة قبل الإصدار للمستخدم.

المسودة المولدة آلياً:

اختلف العلماء المعاصرون في التأمين التجاري على قولين: الأول أنه محرم لما فيه من الغرر والقمار، وهو رأي جمهور المجامع الفقهية. والثاني أنه جائز للحاجة والضرورة الاجتماعية...

⚙️

خط الأنابيب الوكائلي (Agentic Pipeline)

التسلسل المنطقي لمعالجة أي سؤال فقهي يتم إدخاله في النظام، لضمان أعلى درجات الدقة والأمان الشرعي.

1. Query Analysis

معالجة

تصنيف الموضوع (عبادات/معاملات) · استخراج الكيانات الفقهية · تحديد المذاهب ذات الصلة عبر LLM.

2. HyDE Expansion

توسيع

توليد إجابة افتراضية (Hypothetical Document) للبحث الدلالي لتحسين دقة استرجاع الفتاوى المشابهة بنسبة كبيرة.

3. Vector Search

استرجاع

جلب أفضل 20 نتيجة من قاعدة ChromaDB باستخدام تضمينات عربية عالية الدقة (text-embedding-3-large).

4. Reranking

ترتيب

استخدام نموذج Cross-encoder عربي لإعادة تقييم وترتيب أفضل 5 نتائج بوزن أعلى للمصادر الأصيلة والمعتمدة.

5. Guardrails Check

أمان

كشف المسائل الحساسة/السياسية · حساب نسبة الثقة · اتخاذ قرار التصعيد للمفتي البشري تلقائياً إذا لزم الأمر.

6. Generation & UI

توليد

استخدام GPT-4o مع System Prompt شرعي لصياغة الإجابة الموثقة + عرض المصادر ودرجة الثقة للمستفتي.

Stack التقني المقترح لبناء النظام:

LangGraph (Orchestration) · ChromaDB (Vector Store) · GPT-4o (Generation) · text-embedding-3-large (Embeddings) · FastAPI (Backend)
